# Bitzprice

_Build following [Brad Traversy](https://www.youtube.com/watch?v=IkOVe40Sy0U&t=1s) youtube tutorial for practice Next.js_

## Build Setup

```bash
# instal dependencies
$ yarn

# serve with hot reload at localhost:3000
$ yarn dev

# build for production
$ yarn build

# static export
$ yarn export
```
